# Моделирование сезонности в прогнозировании временных рядов

## Гипотеза
**Как лучше всего моделировать сезонность**  


## Структура проекта
```
.
├── README.md
├── requirements.txt
├── src/
│   └── config.py
├── results/
│   └── analysis_results.ipynb
└── run_experiment.py
└── отчет_о_работе.pdf
```

Эти файлы содержат функции для запуска:
1. Загрузка и сэмплирование M4 Monthly данных 
2. Фильтрация серий с явной сезонностью (на основе ACF)
3. Обучение бейзлайнов: Naive, SeasonalNaive, AutoTheta, AutoETS
4. Обучение CatBoost с 6 вариантами фич
5. Подсчет метрик и сохранение в папку `results/`
